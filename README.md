# Clean Architecture with Element-Plus

## Techs
- VueJS 3
- Pinia
- Vite
- TailwindCSS
- Axios
- Typescript
- Json-Server
- Json-Server-Auth

## Running project

```bash
# Clone this repo HTTPS
$ git clone https:github.com/desafio-OM30.git

# Clone this repo SSH
$ git clone git@github.com:[usuario]/desafio-OM30.git

# start
$ yarn dev

# Run fake backend
$ yarn server
```
